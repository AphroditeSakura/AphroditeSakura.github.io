如何在Windows上使用hexo博客框架搭建博客

Hexo简介

Hexo是一款基于 Node.js 的静态博客框架，该框架依赖少且易于安装使用，可以方便的生成静态网页托管在GitHub。使用 Markdown（或其他渲染引擎）解析文章，即可利用靓丽的主题生成静态网页。

Hexo搭建步骤

  1. 安装Git
  2. 安装Node.js
  3. 安装Hexo
  4. 在GitHub创建个人仓库
  5. 将hexo部署到Github

安装Git

首先去官网下载git for windows：下载链接，但国内官网下载可能有点慢，嫌慢的可以选择淘宝的镜像：下载链接，选一个自己需要的版本下载好之后打开安装包安装就可以了，安装完毕打开命令提示符（Win + R)， 检查Git是否安装成功

git --version
